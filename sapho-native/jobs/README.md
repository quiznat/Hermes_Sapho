# Job Templates

This directory stores bounded job templates.

- Personas live in `personas/`.
- Jobs live here.
- Gates may also live here when they are promptable procedures rather than personas.

In the Daily rail:

- `curator/admission.md` is a Curator job
- `extractor/evidence.md` is an Extractor job
- `synthesist/article-claims.md` is a Synthesist job
- `synthesist/article-write.md` is a Synthesist job
- `synthesist/daily.md` is a Synthesist job
- `conclave/daily-gate.md` is a Conclave gate template
