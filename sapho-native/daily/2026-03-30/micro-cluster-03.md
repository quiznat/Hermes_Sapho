## Cluster Thesis

These four papers converge on a shared premise: software engineering agents require carefully engineered infrastructure—not just stronger base models—to achieve reliable, high-performance autonomous operation. Each proposes a distinct architectural intervention (knowledge codification, balanced SDK design, custom agent-computer interfaces, and adaptive context management) and validates it through substantial benchmark gains, demonstrating that the path to capable coding agents runs through systems-level innovation.

## Shared Signals

- Performance gains derive from architectural and interface design rather than model scale alone
- Each framework validates through rigorous benchmarking on established software engineering task suites
- Modular, extensible architectures enable sustained agent performance and practical deployment

## Article Notes

- Codifying Expertise: A Software Engineering Framework for Knowledge-Augmented AI Agents — Combines executable procedural rules with LLM augmentation to capture and embed human domain expertise, achieving 206% output quality improvement and expert-level ratings in an industrial visualization case study.
- Confucius Code Agent: Open-Source AI Engineering at Industrial Scale — Achieves 54.3% Resolve@1 on SWE-Bench-Pro through a three-axis design balancing agent reasoning, user transparency, and developer experience, closing the gap with proprietary alternatives while maintaining full reproducibility.
- SWE-agent: Agent-Computer Interfaces Unlock Automated Software Engineering — Demonstrates that purpose-built agent-computer interfaces are a major performance lever, delivering a 10.7 percentage point gain on SWE-bench Lite by redesigning commands, feedback, and guardrails around coding work.
- ACE: Agentic Context Engineering for Adaptive LLM Performance — Prevents context collapse through structured generation, reflection, and curation, achieving +10.6% on agent tasks and +8.6% on finance benchmarks through unsupervised learning from natural execution feedback.
