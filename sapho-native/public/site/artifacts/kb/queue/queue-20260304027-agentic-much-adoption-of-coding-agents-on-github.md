## Core thesis
A large-scale study provides empirical evidence on coding agent adoption through analysis of real-world GitHub artifacts, quantifying adoption rates and examining commit characteristics to ground understanding of AI-assisted development in production data.

## Why it matters for Sapho
Measuring actual adoption patterns enables developers, researchers, and organizations to assess how coding agents integrate into software engineering workflows, substituting anecdote with evidence-based insight about the pace of change.

## Key findings
- Analysis draws on real-world software engineering artifacts and production data from GitHub.
- Quantitative measures reveal concrete adoption rates and qualitative patterns in commit characteristics.
- The study, authored by Robbes, Matricon, Degueule, Hora, and Zacchiroli, was submitted January 26, 2026.

## Limits
