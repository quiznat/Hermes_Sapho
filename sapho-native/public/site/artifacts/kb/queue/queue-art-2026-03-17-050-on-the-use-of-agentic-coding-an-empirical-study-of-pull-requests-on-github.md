## Core thesis
Agent-assisted pull requests generated by Claude Code achieve an 83.8% merge rate with median merge speeds comparable to human PRs (1.23 vs 1.04 hours), demonstrating practical viability for open-source development while highlighting that human review remains essential for bug fixes, documentation, and complex codebases.

## Mechanism summary
The study empirically analyzed 567 GitHub pull requests generated by Claude Code across 157 diverse open-source projects. Results show agents excel at refactoring, tests, and documentation, with 54.9% of merged PRs requiring no human modification. However, 45.1% of merged PRs required bug fixes by reviewers, 27.4% required documentation changes, and agents still struggle with legacy-heavy codebases and cross-cutting PRs. Rejections were driven by alternative solutions, oversized scope, or obsolescence rather than inherent code quality issues.

## Why it matters for Sapho
This matters because it provides the first large-scale empirical validation of agentic coding in real-world open-source environments, moving beyond demo scenarios to production GitHub workflows. The high merge rate demonstrates that current agent capabilities are already viable for substantial development tasks, while the detailed breakdown of where humans still need to intervene—particularly bug fixes and complex architectural changes—provides actionable guidance for teams integrating AI agents into their development pipelines.
