## Core thesis
Researchers have demonstrated that domain-general automatic evaluators can significantly improve digital agent performance on web navigation and device control tasks, achieving a 29% performance gain on WebArena without requiring additional human supervision or labeled training data.

## Why it matters for Sapho
As digital agents become more capable of autonomously navigating complex interfaces, the bottleneck shifts from task execution to reliable evaluation. Automatic evaluators that generalize across domains offer a scalable path to self-improvement, reducing dependence on expensive oracle feedback while maintaining practical trade-offs between inference cost, modularity, and accuracy.

## Key findings
- Evaluators achieved 74.4–92.9% agreement with oracle metrics across popular digital agent benchmarks, demonstrating strong alignment with ground-truth assessments.
- State-of-the-art performance improved by 29% on WebArena and by 75% in a challenging domain transfer scenario, showing robust generalization.
- The approach balances practical deployment constraints by exploring trade-offs between inference cost, system modularity, and evaluation accuracy.
- All code and data have been released to support reproducibility and further research.
