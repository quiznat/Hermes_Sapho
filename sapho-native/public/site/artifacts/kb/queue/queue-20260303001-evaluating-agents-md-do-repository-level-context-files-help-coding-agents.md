## Core thesis
This paper presents a systematic empirical evaluation of repository-level context files (AGENTS.md) for coding agents, measuring their impact on task completion rates and inference costs through controlled experiments.

## Why it matters for Sapho
Context files are widely adopted to guide coding agents, yet their actual effectiveness has remained largely unverified. Rigorous measurement of both performance gains and computational overhead helps practitioners decide whether maintaining these files is worth the investment.
