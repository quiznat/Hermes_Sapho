## Core thesis
Researchers have introduced SWE-ContextBench, a new benchmark designed to evaluate how effectively programming agents learn from and reuse prior experience when solving software engineering tasks.

## Why it matters for Sapho
As AI coding agents move from isolated task execution to ongoing collaborative tools, their ability to retain and apply context from past interactions becomes a critical measure of practical utility. Testing this capability requires rigorous evaluation frameworks that go beyond single-shot performance metrics.

## Key findings
- The paper introduces SWE-ContextBench, the first benchmark targeting experience reuse in programming agents.
- Findings demonstrate the benchmark's effectiveness at evaluating context learning capabilities.
- The work establishes a framework for assessing how coding agents accumulate and apply knowledge across sessions.
