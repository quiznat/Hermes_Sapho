## Core thesis
AI coding agents have matured into three distinct, non-interchangeable categories—editor assistants, autonomous agents, and orchestration layers—each serving a specific role in the development lifecycle rather than competing for the same use case.

## Why it matters for Sapho
Understanding these categories helps developers and teams choose the right tool for their workflow—whether they need inline assistance, repository-wide automation, or coordinated multi-agent infrastructure—avoiding mismatched expectations and wasted investment.

## Key findings
- Editor assistants operate within IDEs to enhance line-by-line code writing, providing immediate contextual help during development.
- Autonomous agents such as Claude Code and Cursor work at the repository level, capable of executing multi-file changes, running tests, and iterating without constant human guidance.
- The orchestration layer coordinates multiple agents and manages sandboxed execution environments, representing the infrastructure tier that enables complex agent collaboration.
