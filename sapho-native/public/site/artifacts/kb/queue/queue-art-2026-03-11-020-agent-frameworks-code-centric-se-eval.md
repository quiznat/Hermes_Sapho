## Thesis

Agent-framework performance on code-centric software engineering is currently moderate and highly trade-off-driven; framework selection materially affects effectiveness, efficiency, and cost outcomes by task type.

## Mechanism summary

The paper runs a comparative empirical evaluation of seven general-purpose agent frameworks across three code-centric tasks (software development, vulnerability detection, program repair), using task-specific established benchmarks and analyzing outcomes over effectiveness, efficiency, and token/cost overhead.
