## Thesis

General-purpose LLM agents can gain materially more reliable domain execution when workflows are constrained by explicit pseudocode-style SOP decision graphs instead of free-form model-driven reasoning.

## Mechanism summary

The paper introduces SOP-agent, which compiles natural-language SOPs into traversable decision graphs to guide long-horizon task execution. Evaluation spans multiple domains (decision-making, search/reasoning, code generation, data cleaning, grounded customer service) and compares against general-purpose and domain-specific agent baselines, alongside a new Grounded Customer Service Benchmark for SOP-conditioned agent decisions.

## Why retain

This is a primary empirical research paper with explicit cross-domain benchmarking and a new benchmark artifact, directly aligned with queue priorities around agent workflow reliability and structured orchestration.
