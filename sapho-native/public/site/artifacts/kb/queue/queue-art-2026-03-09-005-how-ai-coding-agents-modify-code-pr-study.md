## Thesis

AI coding agents are now producing pull requests at meaningful scale, and their code-change profile differs materially from human contributors across commit structure and diff characteristics.

## Mechanism summary

The paper analyzes merged pull requests from the AIDev dataset and compares agentic PR behavior with human PR behavior on additions, deletions, commits, and files touched, then measures alignment between PR descriptions and code diffs via lexical and semantic similarity.
