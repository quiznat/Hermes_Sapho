# AGENTS.md Boosts AI Coding Agent Efficiency

## Core Thesis

Researchers from Singapore Management University, Heidelberg University, University of Bamberg, and King's College London have quantified how AGENTS.md files affect AI coding agent performance, demonstrating measurable impacts on both runtime efficiency and token consumption when agents process GitHub pull requests.

## Why It Matters

As AI coding agents like Codex and Claude Code become standard tools in software repositories, understanding which documentation practices optimize their efficiency directly affects development velocity and operational costs for engineering teams managing code at scale.

## Key Findings

- AGENTS.md files demonstrate measurable impact on AI coding agent efficiency across tested scenarios.
- Token usage and wall-clock time-to-completion serve as reliable metrics for evaluating agent performance.
- The study employs rigorous experimental design incorporating real-world GitHub pull request data.
