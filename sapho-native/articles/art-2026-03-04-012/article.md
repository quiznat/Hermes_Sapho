# Mapping the Failure Modes of Multi-Agent LLM Systems

## Core Thesis

A team of researchers from UC Berkeley and collaborating institutions has developed MAST, a comprehensive taxonomy, and MAST-Data, a dedicated dataset, to systematically categorize and analyze why multi-agent LLM systems fail. This framework provides the first structured approach to understanding breakdown patterns across different models and tasks.

## Why It Matters

As multi-agent LLM systems increasingly power complex applications—from collaborative coding to automated research—their reliability becomes critical. Without a common language for failure modes and standardized data for comparison, the field lacks the foundation needed to build more robust systems. This work establishes that baseline.

## Key Findings

- Introduces MAST-Data and the MAST taxonomy, creating the first systematic framework for cataloging multi-agent LLM failures
- Analyzes failure patterns across multiple models and task types, revealing how and where these systems break down
- Outlines a research roadmap with concrete insights for improving multi-agent system reliability
