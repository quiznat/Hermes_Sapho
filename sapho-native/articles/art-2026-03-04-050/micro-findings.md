* Moltbot, an open-source, self-hosted AI personal assistant, has been renamed to OpenClaw as of January 30, 2026.
* Moltworker is a new solution that allows running Moltbot on Cloudflare's Sandbox SDK and Developer Platform APIs, enabling it to run online without dedicated hardware.
* Cloudflare Workers' runtime now has significantly improved Node.js compatibility, natively supporting APIs that previously required mocking.
