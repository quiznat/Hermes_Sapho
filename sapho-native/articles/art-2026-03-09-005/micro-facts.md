- After filtering, the analysis used 24,014 Agentic PRs with 440,295 commits with valid patches and 5,081 Human PRs with 23,242 commits with valid patches.
- Human PR commit metadata and file-level patches were reconstructed from the GitHub REST API because the AIDev Human PR set lacked commit-level information.
- Files touched were counted as unique modified file paths across all commits in a PR, and GitHub-reported file renames were treated as single-file modifications.
- The largest reported practical difference between Agentic and Human PRs was commit count, with Cliff's delta = 0.5429, labeled a large effect.
- Files touched differed between Agentic and Human PRs with Cliff's delta = 0.4487, labeled a medium effect.
- Deleted lines differed between Agentic and Human PRs with Cliff's delta = 0.4462, labeled a medium effect.
- Added lines differed between Agentic and Human PRs with Cliff's delta = 0.2836, labeled a small effect.
- Total line changes (additions plus deletions) differed between Agentic and Human PRs with Cliff's delta = 0.3158, labeled a small effect.
- The Mann-Whitney U tests reported statistically significant differences between Agentic and Human PRs across the examined structural metrics at p ≤ 0.001.
- Agentic PRs had slightly higher central tendency than Human PRs across all four description-to-diff alignment metrics reported in Table 2.
- Median semantic similarity was higher for Agentic PRs than Human PRs for both models: CodeBERT 0.9375 vs 0.9347 and GraphCodeBERT 0.8302 vs 0.8067.
- Okapi BM25 scores in this study are unbounded, showed extremely high variance, and the authors state they should be treated only as a relative lexical signal rather than a calibrated similarity measure.
